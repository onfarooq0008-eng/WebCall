# ChatCall WebRTC App

Deploy-ready 1-to-1 room calling app with:
- WhatsApp-like UI
- room link sharing
- video and audio calling
- chat messaging
- Socket.IO signaling

## Local run
```bash
npm install
npm start
```

Open:
`http://localhost:3000`

## Koyeb
- Build command: `npm install`
- Start command: `npm start`
- Port: `3000` (or use the platform `PORT` env var)
- Use HTTPS on the deployed domain

## Important
For best call reliability on mobile networks or strict firewalls, add a TURN server later.
